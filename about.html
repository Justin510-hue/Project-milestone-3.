About Me and This Project

My name is Justin Ramsey, and I am an Information Technology student at Kennesaw State University. This website was created as part of my IT 3203 Web Development course.

The project gave me the opportunity to research HTTP while also practicing important web development skills such as HTML, CSS, JavaScript, responsive design, website organization, and GitHub Pages deployment.

About the Student

I am currently pursuing a Bachelor of Science in Information Technology at Kennesaw State University. I am interested in learning how websites, networks, applications, and computer systems work together.

Web development is an important area of information technology because websites are used for communication, education, business, entertainment, and many other services. This project helped me understand both the technical and design sides of building a website.

Why I Chose HTTP

I chose HTTP because it is one of the main technologies that makes the World Wide Web possible. Most people use HTTP or HTTPS every day without thinking about what happens behind the scenes.

Learning about HTTP helped me understand how browsers request information, how servers respond, how status codes work, and why secure connections are important.

Purpose of the Website

The purpose of this website is to explain HTTP in a way that is organized, understandable, and useful for students who are beginning to learn about web development.

The website covers the history of HTTP, the request-and- response process, common methods, status codes, HTTPS, cookies, sessions, caching, URLs, DNS, and APIs.

Project Goals

  • Explain HTTP using clear and understandable language

Challenges During Development

One challenge was making sure every page used the same navigation, colors, spacing, and overall design. Another challenge was making the layout work properly on both large and small screens.

I also had to make sure image paths, hyperlinks, CSS files, and JavaScript files were connected correctly. Small errors in a filename or folder location could prevent part of the website from working.

What I Learned

Through this project, I learned that building a website requires more than writing code. It also requires planning, testing, research, organization, and attention to detail.

I gained a better understanding of responsive web design and learned how HTML, CSS, and JavaScript work together to create a complete interactive website.
